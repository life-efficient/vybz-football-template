# Vybz Football Template

Editable Next.js template for the Vybz Football website.

## Edit the site

- Main page content: `src/app/page.tsx`
- Global colors and typography: `src/app/globals.css`
- Page title and description: `src/app/layout.tsx`
- Photos and logo: `public/images`

Most repeated sections use arrays near the top of `src/app/page.tsx`, including nav items, pillars, schedule entries, player levels, pathway steps, and social gallery images.

## Run locally

```bash
npm install
npm run dev
```

## Check before publishing

```bash
npm run lint
npm run build
```
